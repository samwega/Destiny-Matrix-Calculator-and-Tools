How to Use the "NOT" Search

You can exclude dates that have a specific number in any of the four main fields (Core, Purpose, Social, Destiny).

To do this, simply add a dash (a minus sign, "-") before the number you want to exclude.

  • Example: If you want to find dates where the Core number is not 11, you would enter -11 into the "Core #" field.
  • This will show you all dates in the selected range that do not have 11 as their Core number.

You can use this for any of the four fields, and you can even combine it with other searches.
